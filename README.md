# ReachFive Identity Web UI SDK

## Installation

The following command installs the Identity Web UI SDK as a Node.js dependency:

```sh
npm install --save @reachfive/identity-ui
```

## Testing

To launch the tests, run:

```
npm run test
```

To update the tests snapshots:

```
npm run test:update
```
